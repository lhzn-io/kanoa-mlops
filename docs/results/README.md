## Contributing

See the [Contributing Guide](../source/contributing.md) for instructions on submitting your benchmark results.

## Result Format

Each result file should follow the template in the Contributing Guide and include:

- Hardware specifications
- Software versions
- Model configuration
- Performance metrics (throughput, latency, memory usage)
- Raw benchmark data

## Hardware Coverage

We're especially interested in results from:

- **NVIDIA Jetson** (Orin, Thor, etc.)
- **Consumer GPUs** (RTX 40/50 series)
- **Data Center GPUs** (A100, H100, etc.)
- **AMD GPUs** (MI series, consumer)
- **Apple Silicon** (M-series)
